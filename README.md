# nestjs-graphql-relay
Example backend server for relay :
- NestJS 6
- Relay Style
  - Identification, Connections
  - using relay-js
- Code first approach
  - using type-graphql
- TypeORM


## Running the app

```bash
# start mysql server
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev
```

Go to `http://localhost:3000/graphql`
